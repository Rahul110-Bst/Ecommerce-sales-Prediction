# E-commerce Customer Churn Prediction

## 📌 Project Overview

Customer churn is one of the biggest challenges for subscription-based and e-commerce businesses. This project predicts whether a customer is likely to leave the company based on demographic information, account details, and service usage.

The project follows a complete end-to-end Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training, pipeline creation, and deployment using Streamlit.

---

## 🎯 Objective

Build an end-to-end Machine Learning application that:

* Predicts customer churn.
* Provides churn probability.
* Uses a Scikit-learn Pipeline for automated preprocessing and prediction.
* Deploys the model with an interactive Streamlit web application.
